Constructor (Blueprint)

  • Constructor act as an blueprint of an object.
  • Object: Person
  • Property of a person: first name, last name, age, eye colour, nationality
  • So, blueprint of person is it must have first name, last name, age eye colour and nationality
  • Person is father, mother, brother, etc
  • Notes: its a good practice to start an object name with Capital letter

Since we can not add a new property to an existing object constructor, we use "Object.prototype"

Object Iterables

Iterable objects are objects that can be iterated over with for..of.

Objects can be iterable using for..of if its:

  • String
  • Array
  • Sets
  • Maps

We will face problem if we want to customer iterable.Therefore, we need to use:

  • Symbol.iterator
  • next()
  • value
  • done


Sets

Sets doesnt hold key value (Maps does)

Steps to create Sets:

  • Passing an Array to new Set() constructor
  • Create a new Set and use add() to add values
  • Create a new Set and use add() to add variables


Maps

Maps holds key value

Steps to create Maps:

  • Passing an Array to new Map() constructor
  • Create a Map and use Map.set()


Functions

In JS, Functions is an object.

Object with method(function) that has arguments.

Call()

Using "call()" method for object's function that accept argument.

Apply()

"apply()" is almost the same as "call()" but it take arguments in array

Useful when deals with number!

Bind()

"bind()" is use to preserve "this" because when function is used as a callback, "this" is lost.


Class Inheritance

Consist of:

  • Extends
  • Getter and Setter

Extends

Use the "extends" keyword to inherit all methods from another class.

The super() method refers to the parent class.

By calling the super() method in the constructor method, we call the parent's constructor method and gets access to the parent's properties and methods.

Getter and Setters

The name of the getter/setter method cannot be the same as the name of the property

Best practice is to add "_" before the property name to seperate the getter/setter from the actual property.

Static

You cannot call a static method on an object, only on an object class.

If you want to use the myCar object inside the static method, you can send it as a parameter

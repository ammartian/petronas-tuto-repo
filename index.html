Constructor (Blueprint)

  • Constructor act as an blueprint of an object.
  • Object: Person
  • Property of a person: first name, last name, age, eye colour, nationality
  • So, blueprint of person is it must have first name, last name, age eye colour and nationality
  • Person is father, mother, brother, etc
  • Notes: its a good practice to start an object name with Capital letter

Since we can not add a new property to an existing object constructor, we use "Object.prototype"

Object Iterables

Iterable objects are objects that can be iterated over with for..of.

  • String
  • Array
  • Sets
  • Maps

Extends

Use the "extends" keyword to inherit all methods from another class.

The super() method refers to the parent class.

By calling the super() method in the constructor method, we call the parent's constructor method and gets access to the parent's properties and methods.

Getter and Setters

The name of the getter/setter method cannot be the same as the name of the property

Best practice is to add "_" before the property name to seperate the getter/setter from the actual property.
